# Analytics Portfolio

Hi, I’m Anna — a junior product/data analyst.  
This repository is a collection of my analytics case studies and portfolio projects.

Here I showcase projects that involve:
- SQL data extraction,
- Python-based preprocessing and analysis,
- exploratory data analysis,
- business-oriented metric interpretation,
- dashboard design and reporting.

## Projects

### 1. [EdTech Course Renewal Analysis](https://github.com/anxsel/analytics-portfolio/tree/main/edtech-course-renewal-analysis)
**Directory:** `edtech-course-renewal-analysis`

An end-to-end analytics case for an EdTech product focused on course renewals and student segmentation.

**What was done:**
- extracted student-course data with SQL,
- validated and preprocessed the dataset in Python,
- assigned students to enrollment waves,
- designed a dashboard for renewal monitoring and retention-focused analysis.

**Key topics:**
- renewal analytics
- student segmentation
- data quality checks
- wave assignment
- BI dashboarding

### 2. [Hotel Booking Survey Analysis](https://github.com/anxsel/analytics-portfolio/tree/main/hotel-booking-survey-analysis)
**Directory:** `hotel-booking-survey-analysis`

An analytical case focused on survey results and customer feedback patterns in the hotel booking domain.

**What was done:**
- explored survey response structure,
- analyzed detractor / passive / promoter segments,
- compared factor distributions across periods,
- identified patterns in customer feedback and satisfaction drivers.

**Key topics:**
- customer feedback analysis
- segmentation
- survey analytics
- factor-level comparison
- insight generation
